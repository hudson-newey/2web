# 2Web Example Project

## Installing dependencies

We recommend using Deno to manage dependencies.

This is because deno has a very rich and vibrant
[standard library](https://jsr.io/@std) and ecosystem.

```sh
$ deno install
>
```

## Starting your project

```sh
$ 2web serve
>
```

## Building your project

```sh
$ 2web build
>
```

## Testing your project

```sh
$ 2web test
>
```
